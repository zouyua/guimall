### 提交订单详细流程

```java
submitOrder(SubmitOmsOrderReqVO reqVO) {
    // 1. 验证参数
    验证会员ID
    
    // 2. 查询购物车
    查询会员购物车商品列表
    
    // 3. 计算金额
    遍历购物车商品
    计算总金额 = Σ(商品价格 × 数量)
    
    // 4. 生成订单号
    orderSn = "SN" + 时间戳 + "_" + 会员ID
    
    // 5. 创建订单
    插入订单主表 (oms_order)
    - 会员ID
    - 订单号
    - 总金额
    - 收货地址信息
    - 订单状态 = 0 (待支付)
    
    // 6. 创建订单明细
    遍历购物车商品
    插入订单明细表 (oms_order_item)
    - 订单ID
    - 商品信息
    - 数量、价格
    
    // 7. 清空购物车
    删除会员购物车数据
    
    // 8. 返回订单信息
    返回订单ID、订单号、金额等
}
```

## 存在的问题

### 1. 并发安全问题 ⚠️

**问题描述**:
- 库存扣减没有并发控制
- 可能出现超卖

**解决方案**:
```java
// 使用乐观锁
UPDATE pms_product 
SET stock = stock - #{quantity}
WHERE id = #{id} AND stock >= #{quantity}

// 或使用分布式锁
RedisLock lock = redisLock.lock("product:" + productId);
try {
    // 扣减库存
} finally {
    lock.unlock();
}
```

### 2. 事务一致性问题 ⚠️

**问题描述**:
- 订单创建和优惠券使用不在同一事务
- 可能导致数据不一致

**解决方案**:
```java
@Transactional(rollbackFor = Exception.class)
public Response submitOrder(SubmitOmsOrderReqVO reqVO) {
    // 1. 创建订单
    // 2. 使用优惠券
    // 3. 扣减库存
    // 4. 清空购物车
    // 确保全部成功或全部回滚
}
```

### 3. 优惠券未集成 ⚠️

**问题描述**:
- 订单提交时没有使用优惠券
- couponId、couponAmount 字段为空

**解决方案**:
```java
// 在提交订单时
if (reqVO.getCouponHistoryId() != null) {
    // 1. 验证优惠券
    // 2. 计算优惠金额
    // 3. 使用优惠券
    // 4. 更新订单优惠信息
}
```

### 4. 库存未扣减 ⚠️

**问题描述**:
- 提交订单时没有扣减商品库存
- 可能导致超卖

**解决方案**:
```java
// 在提交订单时
for (OmsCartItemDO item : cartItems) {
    // 检查库存
    PmsProductDO product = productMapper.selectById(item.getProductId());
    if (product.getStock() < item.getQuantity()) {
        throw new BizException("库存不足");
    }
    
    // 扣减库存
    int updated = productMapper.decreaseStock(
        item.getProductId(), 
        item.getQuantity()
    );
    if (updated == 0) {
        throw new BizException("库存扣减失败");
    }
}
```

### 5. 缺少订单超时取消 ⚠️

**问题描述**:
- 未支付订单没有超时自动取消机制
- 会占用库存

**解决方案**:
```java

@Scheduled(cron = "0 */5 * * * ?") // 每5分钟执行
public void cancelTimeoutOrders() {
    // 查询30分钟未支付的订单
    // 取消订单
    // 回滚库存
    // 回滚优惠券
}
```
